// Case studies

Playwright framework, built from zero Northwestern Mutual

Dec 2024 – Present

There was no automated test coverage across 30+ internal microservices and integrations when I joined. I designed the framework architecture using Page Object Model for maintainability, built reusable components, and wired it into our GitLab CI pipeline.

Result: release validation time cut by 40%, with tests gating CI/CD deploys instead of running as an afterthought.

Pre-WFI recon routing bug Caught via exploratory testing

Northwestern Mutual

Recon-created cases for old policies were being misrouted to the legacy system instead of the new one. Root cause: those cases lacked workflow-initiator (WFI) data — which only exists for cases created through the live workflow — so the system misclassified old policies as new ones and sent them down the wrong path.

I traced this to the missing data signal and worked with engineering to get it corrected before it affected production billing.

Shift-left testing on a government billing platform Peraton / County of San Diego

Mar 2020 – Oct 2024

Owned QA for the County's property tax system — a platform where a bug means a resident's bill is wrong. Introduced shift-left practices so 60% of defects were caught at the API/integration layer before UI testing even started, and reduced manual testing effort by 50% through automation-first strategies.
